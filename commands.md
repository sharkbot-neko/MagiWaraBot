# コマンド一覧

```
基本的なコマンド
!help .. コマンド一覧を表示します。
!omikuji ..おみくじを引きます。
!test .. テストを実行します。

検索系のコマンド
!roominfo .. ルーム情報を取得します。
!lookup .. ユーザー情報を取得します。
!mid .. 実行した人のmidを取得します。
!translate .. 翻訳をします。
!wiki .. Wikipediaを検索します。

経済系のコマンド
!work .. コインを稼ぎます。
!money .. コイン残高を確認します。

グループオーナーのみが使えるコマンド
!ban .. メンバーをbanします。
!unban .. メンバーをunbanします。
!welcome .. 新しい人が参加するとお知らせします。
```
